# cyber.nvim

A slightly modified port of [AstroNvim](https://docs.astronvim.com/)


## Pre-requisites

Everything listed [in the docs](https://docs.astronvim.com/#-requirements), plus:

- Node/npm
- Go
- Python
- [dwt1/shell-color-scripts](https://gitlab.com/dwt1/shell-color-scripts)

## Usage behind a firewall/corporate proxy

Switch to branch `no-concurrency`
